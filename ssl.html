SSL Certificate Upload

Paste your PEM-encoded certificate and private key directly in the fields below. We’ll write them to the keys folder used by nginx so you don’t have to SSH into the server.

Use full PEM blocks including BEGIN CERTIFICATE and BEGIN PRIVATE KEY lines. On save, files are written to the shared keys directory (default). Restart or reload nginx to apply.

Waiting to upload…
After uploading, reload the nginx container or process to start using the new certificate.
